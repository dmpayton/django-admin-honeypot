============
Installation
============

Requirements
============

* Python 2.6+ or Python 3.3+
* Django 1.4+

Install
=======

django-admin-honeypot is on `PyPI`_ and can be installed with `pip`_:

::

    pip install django-admin-honeypot

.. _PyPI: http://pypi.python.org/
.. _pip: http://www.pip-installer.org/
